# Project Constitution

## Core Identity
- **Project**: Lichess Tournament Filter
- **Type**: Chrome Extension
- **Language**: TypeScript (Strict Mode)

## Absolute Rules
1.  **Simplicity**: Code must be concise and focused. No over-engineering.
2.  **Performance**: The extension must be lightweight and fast.
3.  **Aesthetics**: UI elements must blend seamlessly with Lichess design.
